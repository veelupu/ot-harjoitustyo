## Tämä on tämän työn toinen alaotsikko
 
_Mitä on oppia?_

Joku joskus sanoi viisaasti:

> Uuden oppiminen vaatii lähinnä istumalihaksia.

Toisin ilmaistuna

```java
if (haluaaOppia) {
	istuAlas;
	opiskele;
	return tulokset;
}
```
